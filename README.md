# Sales & Customer Performance Dashboard Using Power BI

An interactive Power BI dashboard built on global retail sales data, designed to give stakeholders a clear, drill-down view of sales performance, customer segments, product categories, and return trends to support data-driven business decisions.

## Project Overview

Raw retail sales data is rarely analysis-ready — this project takes a global retail dataset through cleaning and transformation, models it using a star-schema design, and layers on DAX-driven KPIs and interactive visuals so business stakeholders can self-serve insights without needing to write a single query.

## Tools & Technologies

Power BI, DAX, Power Query, Data Modeling (Star Schema), Data Visualization

## What This Dashboard Covers

- **Data cleaning & transformation:** Used Power Query to clean and reshape global retail sales data into a consistent, analysis-ready format
- **Data modeling:** Built a star-schema data model connecting fact and dimension tables for efficient, scalable reporting
- **DAX measures:** Created calculated measures for Total Sales, Profit, Profit Margin, Return Rate, and Shipping Cost analysis
- **Regional performance analysis:** Visualized sales and profit trends across regions to surface top- and under-performing markets
- **Customer segmentation:** Broke down customer behaviour and value by segment
- **Product category analysis:** Identified top-performing and underperforming product categories
- **Return trend analysis:** Tracked return rates to flag products or categories with quality or fulfillment issues
- **Interactive features:** KPI cards, drill-through reports, slicers, and cross-filtered visuals for self-service exploration
